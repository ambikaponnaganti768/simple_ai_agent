# simple_ai_agent
AI-powered agent developed with Python, LangChain, LangGraph, and Ollama. Demonstrates agent workflows, tool calling, and local LLM deployment using Llama 3.
🤖 AI Agent using LangChain, LangGraph & Ollama

Overview

This project is a beginner-friendly AI Agent built using:

- LangChain
- LangGraph
- Ollama
- Llama 3

The goal of this project is to understand how AI agents work, how tools can be integrated into agent workflows, and how local LLMs can be used without paid API subscriptions.

Features

- Conversational AI Agent
- Calculator Tool
- Knowledge Search Tool
- Local LLM using Ollama
- Tool Integration with LangChain
- Agent Workflow Design
- No OpenAI API Cost

Tech Stack

- Python
- LangChain
- LangGraph
- Ollama
- Llama 3

Installation

1. Clone Repository

git clone <repository-url>
cd ai-agent-project

2. Create Virtual Environment

python -m venv venv

Activate:

venv\Scripts\activate

3. Install Dependencies

pip install langchain
pip install langgraph
pip install langchain-ollama

4. Install Ollama

Download and install Ollama:

https://ollama.com

5. Download Llama 3

ollama pull llama3

6. Run the Agent

python agent.py
